¿Qué es MovieHub?

MovieHub es una aplicación desarrollada con Angular y Bootstrap que consume una API de películas para mostrar información actualizada sobre títulos, géneros, sinopsis y valoraciones.

Los usuarios pueden buscar películas, ver detalles completos y guardar sus favoritas para acceder rápidamente a ellas más adelante.
